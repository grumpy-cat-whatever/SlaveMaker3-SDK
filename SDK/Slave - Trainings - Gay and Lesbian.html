Lesbian/Gay Training
Slave Maker 3 SDK Version 8

There are a few variables and functions for training your slave for same-sex relationships, as a lesbian or as gay.

Please remember the church of the New Gods discourages lesbian acts and considers them immoral. As this is the state religion the XXX contest, a somewhat discouraged event in itself, does not include lesbian events.

The faith of the Old Gods has no restrictions on lesbianism or homosexuality.

Sexuality in the Game

Each slave has a statistic Sexuality that records their sexual orientation. This system is not an attempt to represent realism but a fictional system of sexuality where a person can be encouraged to become lesbian by exposure to lesbian type acts.

For this game the Sexuality stat also represents male homosexuality, so it best represents same-sex interest, not just lesbian. For simplicity I will use the term lesbian from here on, but for male slaves please read homosexual.

Sexuality Scale
< 25   
a lesbian, prefer partners of the same sex and dislikes opposite sex partners
25-40
prefers same sex partners and uncomfortable with opposite sex partnets
41-59
bisexual, no preference of same or opposite sex partners
60-75
mostly heterosexual, uncomfortable with same sex partners
76-100
heterosexual, dislikes same sex partners.

By default all slaves start with a sexuality of 100 to indicate they are heterosexual. The game requires special events to cross the lesbian/heterosexual boundaries of 25/75. Otherwise the game will limit sexuality to being > 25 initially. Once the value drops below 76 it will not then exceed 75. Once lesbian/gay training starts and sexuality drops below 25 it will never then exceed 25.

Reluctance
For each act a variable ReluctanceLevel is set to indicate how uncomfortable the slave is with the current act. This is calculated each time an act is done and is not saved by the game.
0 = no problems with the act
1 = uncomfortable
2 = dislikes the act
3 = hates the act

This is set depending on the sexuality of the slave and the gender of the partners involved. It is only > 0 for sex acts.

If a developer does not want to use this reluctance system then set ReluctanceLevel to -1. The game will treat this as 0 and the slave will not care about same sex acts. In this special case the level is never recalculated.
You must initialised this value in your Initialise() function for actionscript or <Initialise> node for xml. You can also set in an event as desired but remember the value is not saved.

eg
actionscript:
function Initialise() {
    _root.ReluctanceLevel = -1;
     .....
}

XML:
<Initialise>
    <SetVar reluctancelevel='-1'/>
</Initialise>

OR
 in <Slave> node add
<ReluctanceLevel>-1</ReluctanceLevel>

People/NPC's

Tena
Tena, the cute lesbian, is met with walking in the Town Center and at the Docks. When your slave maker personally supervises she can offer to teach your slave maker about training slaves for lesbians if your Lesbian Trainer skill (sLesbianTrainer) is less than 1.
Male slaves cannot visit!

High Priestess Daruna
The high priestess is a lesbian, she conceals this, but it is a sort of open secret, well known just not discussed.

Head Librarian Lilana
Lilana prefers female lovers but does not restrict herself to only females.

Rinno the Shopkeeper
Rinno sells two books that teach aspect of make same-sex love and the basics of gay training. Each increase the gay trainer skill by 0.2, The second book is only available once your skill is 1 or more.

(Boolean)
         true = the slave has decided to undergo same-sex training (lesbian or gay)

LesbianInterest

 lesbianinterest or gayinterest
initialise using <LesbianInterest> or <GayInterest>
(Number) The level of interest your slave has in same-sex training. This is the skill level of sLesbianTrainer or sGayTrainer needed to start training. It is NOT saved so you must set it in Initialise() or UpdateSlave() functions and when it changes
         0 = always will train if sLesbianTrainer or sGayTrainer is 1 or more
         1 = will train is sLesbianTrainer or sGayTrainer is 1 or more
         2 = will train is sLesbianTrainer or sGayTrainer is 2 or more
         3 = will train is sLesbianTrainer or sGayTrainer is 3 or more
Special case
         4 = will never train. Preferably do not use this value, generally set to 3 to indicate high resistance. Only for special cases like Angel Belldandy say

sLesbianTrainer

 lesbiantrainer
(Number) Your Slave Maker or Assistant's skill as a lesbian trainer, values 0-3
         0 = cannot train lesbian slaves
         1 = can train girls who are not hostile to lesbians or religiously opposed. Their lesbian skills are limited to 100 and they will only get the Bi-sexual Ending
         2 = can train most girls, except those with strong objections. Their lesbian skills are limited to 200 and but they can get the Lesbian Ending
         3 = can convince any girl to become a lesbian slave and love it. Their lesbian skills are unlimited and they can get the Joyfull Lesbian Ending

Items

Your slave maker can buy a pair of books from the shop, These are purchased when OtherBooks, bitflag 2, 3 are set. You can also use the IsItemOwned() finction with id 48, 49
eg
 actionscript:
if (_root.OtherBooks.CheckBitFlag(2)) {
   // Volume 1 owned
}
or
if (_root.IsItemOwned(49)) {
   // Volume 2 owned
}

xml:
<if itemowned-48='true;>
   <!-- volume 1 owned -->
</if>

These items are only sold when your Gay Trainer is < 2. The second volume only when the skill is 1 or higher.

They are read by your slave makers once OtherBooks, bitflags 66. 67 are set. They can only be read once and they will happen after night trainings are complete. Your slave can also read the books, and they use the flags 34, 35.

You can intercept the purchase using the slave interface item functions, like PurchaseItem(), AfterTailorYes with item id of 43 (note NOT the same as used in IsItemOwned())

BitFlags

BitFlag 1
         10       = Lesbian/gay training started
         11       = decided to not do lesbian/gay training
         12       = reset = Offer Lesbian/Gay Training next Lesbian act
         27       = start lesbian/gay training at start of day

Note: LesbianTraining is set as
LesbianTraining = BitFlag1.CheckBitFlag(10);

Note: the remaining flags are largely for internal game use

Ending

Ending Number: n/a
Condition:  StartSexuality >= 25 && Sexuality < 25

Images are shown using the interface function ShowEndingLesbianSlave, see below. No default image is shown. This function is used for both male and female cases.

Note no ending number as this ending is prequel to any actual ending.

Core Functions

DefaultLesbian

actionscript:
function DefaultLesbian(chance:Number) : Boolean
XML:
no equivalent

If your slave is in lesbian training then chance% of the time use a generic graphic. Usable for all gender slaves.

 eg
 actionscript:
if (DefaultLesbian(33)) return;


Slave Girl Interface Functions

These functions can be implemented / overloaded for a slave to control her lesbian training.

Note: all of these are only used for same sex female, they are not used for male slaves.

ShowEndingLesbianSlave

actionscript:
function ShowEndingLesbianSlave()
XML:
no specific implementation, the game will show the image
Ending - Lesbian Slave.jpg

Called when the ending for a lesbian slave is shown. Generally show here the image of the slave kissing a woman. Do not show any text.

 eg
 actionscript:
function ShowEndingLesbianSlave() {
   _root.Backgrounds.ShowBedRoom();
   _root.ShowMovie(EndingLesbianSlave, 1, 1);
}

 Note: The game does not show a default image for this ending if you do not implement this function.
