Core Variables

Actionscript Variable XML Variable

DoneCourtesan (READ ONLY)

mplemented in: _root, class Slave via SlaveSkills
 donecourtesan (READ ONLY)
(Boolean)
         true = has completed courtesan training
         false = has not

slCourtesan

mplemented in: _root, class Slave via SlaveSkills
 courtesan
(Number) training as a courtesan (0-100)

Please use ChangeCourtesan() or <ChangeCourtesan> to alter this skill.

HighClassProstitute.CustomFlag

 flag-ladyokyanu-custom
flag-highclassprostitute-custom
(also can use custom1, eg flag-ladyokyanu-custom1)
Number) days remaining to the party with Lady Okyanu. Only valid if _root.HighClassProstitute.CheckBitFlag(32) == true

eg
actionscript: xml:
if (_root.HighClassProstitute.CheckBitFlag(32)) {
    _root.SetText("You remember the party with Lady Okyanu is in " + _root.HighClassProstitute.CustomFlag + " days time");
}		 <if flag-highclassprostitute-32='true''>
   <SetText>The party is dues in #flag-highclassprostitute-custom days</SetText>
</if>

HighClassProstitute.PersonFlag

 flag-ladyokyanu-person
flag-highclassprostitute-person
(Number) state of training
         -1       = training started
         -2       = done first lesson
         -3       = done second lesson
         -4       = done last lesson, training complete

Events

Yes/No
8079 - Start Training yes/no, asked at the end of the part with Lady Okyanu

BitFlags

HighClassProstitute
         32       = Made party offer

eg
if (_root.HighClassProstitute.CheckBitFlag(32)) {
    _root.SetText("You remember the party with Lady Okyanu is in " + _root.HighClassProstitute.CustomFlag + " days time");
}

Ending

Ending Number: 26
Condition:  VarConversation > 99 && VarCharisma > 95 && VarRefinement > 60 && VarFuck > 90 && VarBlowJob > 90 && DoneCourtesan

Images are shown using the interface function ShowEndingCourtesan, see below.

Core Functions

ChangeCourtesan

 implemented in: _root and Slave class

actionscript:
function ChangeCourtesan(val:Number)
XML:
<ChangeCourtesan>val</ChangeCourtesan>

increases courtesan training by val points. You can decrease using a negative number.

 eg
 actionscript:
_root.ChangeCourtesan(5);
_root.ChangeCourtesan(-1);

 xml:
<ChangeCourtesan>5</ChangeCourtesan>
<ChangeCourtesan slave='Tifa'>5</ChangeCourtesan>


Slave Interface Functions

These functions can be implemented/overloaded for a Slave to control their courtesan training.

ShowEndingCourtesan

actionscript:
function ShowEndingCourtesan()
XML:
no specific implementation, the game will show the image
Ending - Courtesan.jpg

Called when the ending for a courtesan slave is shown. Generally show here the image for a courtesan. Do not show any text.

eg
 actionscript:
function ShowEndingCourtesan()
{
   _root.Backgrounds.ShowBar();
   _root.ShowMovie(EndingCourtesan, 1, 3);
}

Note: The game does not show a default image for this ending if you do not implement this function.
