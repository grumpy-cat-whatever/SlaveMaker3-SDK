Faeries
Slave Maker 3 SDK Updated for 3.4.02

There are a series of functions and variables that control faerie events in the game. These are all slave specific, they reset for each new slave being trained. Additionally there are variables and functions to represent the transformation of the slave into a faerie.

Note: in the game text I always use the term faerie not fairy, but function names and variables differ.

Core Variables

All variables below are slave specific and are implemented in: _root and Slave class. The _root version is for the current slave in training. The others can be accessed using the GetSlaveDetails() function
eg
actionscript:
if (_root.FairyXF > 50) {
  // current slave transformation 50+
}
var sd:Slave = _root.GetSlaveDetails("Aeris");
if (sd.FairyXF > 50) {
   // Aeris's fiary trainsformation is 50+
}
xml:
<if fairyxf-aeris='50+>
   <!-- Aeris's fairy trainsformation is 50+ -->
</if>
Actionscript Variable XML Variable

FairyXF

 fairytransformation
fairyxf
(Number) stage of her transformation into a fairy
         -1          = will never transform
         0           = not started
         1-99     = partial
         2000    = fully transformed     
Note: generally you should never modify this variable, use the transformation functions IsFairy() ChangeFairyXF(), but this is provided mainly for xml use to check the transformation of an arbitrary slave (see IsFairy() below)

FaeriesRingOK

 itemowned-16='true'
(Number) Does your slave own the faeries ring, 1 for yes, 0 for no.

xml:
Use the conditional listed
<if itemowned='16'>
    <!-- own the faeries ring -->
</if>

FaeriesRingWorn

 itemworn-16='true'
(Number) Is the slave wearing the faeries ring. Note this is an essential prerequisite for the faerie transformation.

FairyMeeting

 fairymeeting
(Number) the progress of the fairy rescue event, generally a sequential number, but varies depending on choices.

Very seldom would you need to check this variable.


Core Functions

Transformation functions

All the transformation functions are slave specific and are implemented in: _root and Slave class. The _root version is for the current slave in training. The others can be accessed using the GetSlaveDetails() function
eg
actionscript:
_root.ChangeFairyXF(5);   //current slave
var sd:Slave = _root.GetSlaveDetails("Aeris");
sd.ChangeFairyXF(4);   //slave Areis

xml:
<ChangeFairyXF>4</ChangeFairyXF>
<ChangeFairyXF slave='aeris'>4</ChangeFairyXF>

ChangeFairyXF
 implemented in: _root and Slave class
actionscript:
function ChangeFairyXF(val:Number)
or
function ChangeFaerieXF(val:Number)
XML:
<ChangeFairyXF slave='slavename'>val</ChangeFairyXF>
<ChangeFaerieXF slave='slavename'>val</ChangeFaerieXF>
slave attribute defaults to the current slave in training if not specified

alter (increase or decrease) her fairy transformation by val points. No effect if the transformation has not started or is complete.

eg
actionscript:
// current slave
_root.ChangeFairyXF(1);
// arbitrary slave
var sd:Slave = _root.GetSlaveDetails("Maran");
sd.ChangeFairyXF(1);

xml:
current slave
<ChangeFairyXF>1</ChangeFairyXF>
arbitrary slave
<ChangeFairyXF slave='Maran'>1</ChangeFairyXF>

Dress Item positioning functions

These functions are used to position the faerie wings in the main window for the dress images of the slave.

Note: these functions position all types of wings for the slave, including demon wings etc.

XML:
each will list a node, these are implemented in the Dress nodes for the slave
eg
<Dresses>
    <Dress0>
           <Halo x='229.5' y='30.95' rotation='0' width='150' height='54'/>
           <Leash x='177.6' y='57' rotation='0' width='-80' height='120' over='true'/>
           <BitGag frame='4' x='220' y='79' rotation='-10' width='55' height='40'/>
           <NymphsTiara x='216' y='47' rotation='-10' width='50' height='45'/>
           <CatEars frame='1' x='222.25' y='52.55' rotation='0' width='50' height='25' over='true'/>
           <Tail x='152' y='226' rotation='0' width='-150' height='80'/>
           <Wings x='217' y='150' rotation='10' width='-300' height='200'/>
    </Dress0>
</Dresses>
PositionWings
PositionWingLeft
PositionWingRight
actionscript:
function PositionWings(xpos:Number, ypos:Number, rot:Number, wid:Number, hei:Number)
function PositionWingLeft(xpos:Number, ypos:Number, rot:Number, wid:Number, hei:Number)
function PositionWingRight(xpos:Number, ypos:Number, rot:Number, wid:Number, hei:Number)
XML:
<Wings x='xpos' y='ypos' rotation='rot' width='wid' height='hei'/>
<WingLeft x='xpos' y='ypos' rotation='rot' width='wid' height='hei'/>
<WingRight x='xpos' y='ypos' rotation='rot' width='wid' height='hei'/>

Position a set of Faerie wings behind the slave's image in the main window. The dress image must have a transparent background for the wings to be seen. PositionWings places a pair of wings, and the others a left or right wing only.

xpos = horizontal position in the main window, 0-480, but you can exceed these limits to have the image partially out of the window
ypos = vertical position in the main window, 0-336, but you can exceed these limits to have the image partially out of the window
rot = rotate the image clockwise in degrees, < 0 for counter-clockwise
wid = width of the image. Will resize the image, will not preserve aspect ratio
hei = height of the image. Will resize the image, will not preserve aspect ratio

Note: the debugging menu can create these parameters for both actionscript and xml

Slave Interface Functions

These functions can be overloaded / implemented for a slave to control their changes. This function is only ever use for the current slave in training. It is never called for assistants, owned slaves or minor slaves.

These are optional to implement

ShowFaerieTransformation

actionscript:
function ShowFaerieTransformation() : Boolean
XML:
<Other>
   <ShowFaerieTransformation>
No return type implemented here.
Note: when this is called the default images + text are already shown. Use <HideImages> and/or <SetText> to replace if desired.
Note: not called if the actionscript ShowFaerieTranformation() returns true

Show an image and/or text when the fairy transformation completes.
return true if you show an image.
