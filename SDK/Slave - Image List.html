Image Formats
The game supports images of type
  • jpeg - please use extension .jpg
  • png - supports transparent backgrounds
  • gif - DOES NOT support animated gifs. If you want animation convert the image to .swf
  • swf - use this format for any animations

Mandatory

Some images are required for a slave to work properly. Ideally you should provide one image for every type listed, but you must always provide

Assistant image for when they are used as an assistant. For swf this is an image returned from InitialiseAsAssistant()
For xml this is an image
Assistant.jpg

You must provide a single Tired image
Tired 1.jpg

You must provide an image for the Slave Market , named
Slave-xxxxx.png
where xxxxx is your slaves name

You must provide at least one dress for xml this is
Dress Plain.png

You must provide one naked image. For xml this is
