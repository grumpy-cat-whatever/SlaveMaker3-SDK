Puppygirl Training
Slave Maker 3 SDK Version 8

There are a series of functions and variables that control catgirl training for your slaves. Note males can do catboy training but for simplicity I will refer to puppygirls from here on.

Core Variables


Actionscript Variable XML Variable

sPuppyTrainer

implemented in: _root, classes SlaveMaker and Slave via SlaveMakerSkills
 puppytrainer
(Number) your slave makers skill as a puppygirl trainer. Also assistants and slaves have this skill and can be checked via their Slave object
XML can use puppyttrainer-slavename to get the value for an arbitrary slave/assistant. This defaults to your current slave maker.

eg
actionscript: xml:
if ((_root.sPuppyTrainer > 1)
   // slave makers puppy trainer 2 or higher
}
var sd:Slave = _root.GetSlaveDetails("Melisandra");
if (sd.sPuppyTrainer > 2) {
    // Melisandra's puppy trainer 2 or higher
}
if (_root.AssistantData.sPuppyTrainer > 2) {
  // assistants skill 2 or higher
}
 <if puppytrainer='2+'>
   <!-- slave makers puppy trainer 2 or higher -->
</if>
<if puppytrainer-melisandra='2+'>
   <!-- Melisandra's puppy trainer 2 or higher -->
</if>
<if puppytrainer-assistant='2+'>
   <!-- assistants puppy trainer 2 or higher -->
</if>

slPuppyTraining

implemented in: _root, class Slave via SlaveSkills
 puppytraining
(Number) The level of training as a puppygirl for a given slave.
         -1          = will never train
         0           = not started
         1-99     = partial
         2000    = fully trained    
Generally do not directly refer to this variable, use the ChangePuppyTraining(), IsPuppygirl(), IsPuppygirlComplete() or IsPuppygirlTraining() functions.

Note:
The maximum value of this skill is determined by the Slave Makers sPuppyTrainer skill. Level 1 means a maximum of 60, Level 2 is unlimited.

PuppygirlInterest

implemented in: _root, class Slave
 puppygirlinterest
(Number) The level of interest your slave has in being a puppy slave. This is the skill level of sPuppyTrainer needed to start training. It is NOT saved so you must set it in Initialise() or UpdateSlave() functions and when it changes
         0 = always will train if sPuppyTrainer is 1 or more
         1 = will train is sPuppyTrainer is 1 or more
         2 = will train is sPuppyTrainer is 2 or more
Special case
         3 = will never train. Preferably do not use this value, generally set to 2 to indicate high resistance.


BitFlags

Bit Flags 1
         58       = Puppygirl Complete
         57       = started puppygirl training

Ending

Ending Number:
Condition: 

NONE IMPLEMENTED YET

Images are shown using the interface function ShowEndingPuppygirl, see below.

Core Functions

ChangePuppyTraining

 implemented in: _root and Slave class
actionscript:
function ChangePuppyTraining(val:Number)
XML:
<ChangePuppyTraining slave='person'>val</ChangePuppyTraining>

Change puppygirl training by val points. By default the current slave in training is updated.

 eg
 actionscript:
 _root.ChangePuppyTraining(5);
var sd:Slave = _root.GetSlaveDetails("Tifa");
sd.ChangePuppyTraining(5);
xml:
<ChangePuppyTraining>5</ChangePuppyTraining>
<ChangePuppyTraining slave='Tifa'>5</ChangePuppyTraining>

IsPuppygirl

 implemented in: _root and Slave class
actionscript:
function IsPuppygirl() : Boolean
XML:
ispuppygirl

Returns true if your slave is a fully or partially trained puppygirl. Available both via _root (for current slave) or via Slave class.

 eg
 actionscript:
 if (_root.IsPuppygirl()) {
   // current slave is a puppygirl (or training as one
}
var sd:Slave = _root.GetSlaveDetails("Tifa");
if (sd.IsPuppygirl()) {
    // Tifa is a puppygirl
}
xml:
<if ispuppygirl='true'>
    <!-- current slave is a puppygirl -->
</if>
<if ispuppygirl-tifa='true'>
    <!-- Tifa is a puppygirl -->
</if>

IsPuppygirlComplete

 implemented in: _root and Slave class
actionscript:
function IsPuppygirlComplete() : Boolean
XML:
ispuppygirlcomplete

Returns true if your slave is a fully trained puppygirl, returns false if they have not completed training. Available both via _root (for current slave) or via Slave class.

IsPuppygirlTraining

 implemented in: _root and Slave class
actionscript:
function IsPuppygirlTraining() : Boolean
XML:
ispuppygirltraining

Returns true if your slave has started training as a puppygirl but has not yet completed training. Available both via _root (for current slave) or via Slave class.

Dress properties functions

These functions are used to set or check the properties of a slave's dress. Particularly here if the dress includes a cat tail or a set of cat ears.

XML:
each will list a node, these are implemented in the Dress nodes for the slave. The <StatEffects> node has attributes to set these properties.
eg
<Dress6>
    <Name>Cat Suit</Name>
    <StatEffects CatEars='true' CatTail='true'>30, 0, 10, 0, 0, 0, 0, 0, 0, 10, 10, 10, 10, 15, 15, 0, 15, 0</StatEffects>
    <Halo x='220.5' y='24' rotation='-5' width='200' height='50'/>
    <Leash x='272' y='76.5' rotation='0' width='-100' height='140'/>
    <BitGag frame='2' x='233.5' y='113.25' rotation='-10' width='70' height='35'/>
    <NymphsTiara x='222.9' y='46.65' rotation='-10' width='80' height='80'/>

</Dress6>

These functions are used to position the cat-eears in the main window for the dress images of the slave.

Note: the cat-tail is positioned using the generic PositionTail() functions

XML:
each will list a node, these are implemented in the Dress nodes for the slave
eg
<Dresses>
    <Dress0>
           <Halo x='229.5' y='30.95' rotation='0' width='150' height='54'/>
           <Leash x='177.6' y='57' rotation='0' width='-80' height='120' over='true'/>
           <BitGag frame='4' x='220' y='79' rotation='-10' width='55' height='40'/>
           <NymphsTiara x='216' y='47' rotation='-10' width='50' height='45'/>
           <PuppyEars frame='1' x='222.25' y='52.55' rotation='0' width='50' height='25' over='true'/>
           <Tail x='152' y='226' rotation='0' width='-150' height='80'/>
           <Wings x='217' y='150' rotation='10' width='-300' height='200'/>
    </Dress0>
</Dresses>
PositionPuppyEars
actionscript:
function PositionPuppyEars(gframe:Number, xpos:Number, ypos:Number, rot:Number, wid:Number, hei:Number)
XML:
<PuppyEars x='xpos' y='ypos' rotation='rot' width='wid' height='hei' frame='num'/>

Position the puppy ears behind the slave's image in the main window. The dress image must have a transparent background for the ears to be seen.

gframe = select a particular version of the ears. A White/pink and Black/white are the standard set.
xpos = horizontal position in the main window, 0-480, but you can exceed these limits to have the image partially out of the window
ypos = vertical position in the main window, 0-336, but you can exceed these limits to have the image partially out of the window
rot = rotate the image clockwise in degrees, < 0 for counter-clockwise
wid = width of the image. Will resize the image, will not preserve aspect ratio
hei = height of the image. Will resize the image, will not preserve aspect ratio

Note: the debugging menu can create these parameters for both actionscript and xml

PositionPuppyEarsOver
actionscript:
function PositionPuppyEarsOver(gframe:Number, xpos:Number, ypos:Number, rot:Number, wid:Number, hei:Number)
XML:
<PuppyEars x='xpos' y='ypos' rotation='rot' width='wid' height='hei' over='true'/ >

Position a set of puppy ears over top of the slave's image in the main window.
Otherwise identical to PositionPuppyEars.


Slave Interface Functions

These functions can be overloaded for a slave to control their puppygirl training.


These are optional

StartPuppygirlTraining

actionscript:
function StartPuppygirlTraining() : Boolean
XML:
<Other>
    <StartPuppygirlTraining>

Start puppygirl training, show an image and optionally specific text
return true if you show an image, otherwise a default will be shown

ShowRefusePuppygirl

actionscript:
function ShowRefusePuppygirl(type:Number)
XML:
<Other>
    <RefusePuppygirl>
xml variable general is set to the value of type.

type
         29 = wear puppy ears
         30 = wear puppy tail
         1 to 6 = put on dress x with puppy ears or tail

Show an image/text if the slave refuses to start puppygirl training
