Background Images
Slave Maker 3 SDK Updated to version 3.4

There are a group of standard movie clips that show backgrounds behind the main window

Show these by using the functions below. All parameters and attributes are optional.

 actionscript:
_root.Backgrounds.ShowAlley();
or
_root.Backgrounds.ShowAlley(2);
or
_root.Backgrounds.ShowAlley(2, true);

 xml:
<Backgrounds place='alley'/>
or
<Backgrounds place='acove' flip='true'/>

Common Parameters/Attributes for all functions:

flip
    - this will flip the image horizontally, defaults to false if it is not provided. For xml specify flip='true'
eg
actionscript:
_root.Backgrounds.ShowAlley(2, true);
 xml:
<Backgrounds place='alley' flip='true'/>

frame
    - this is a specific image to show, where there are multiple variations for a location. If not specified a random image for the place is shown. For xml specify frame='1' or similar.
eg
actionscript:
_root.Backgrounds.ShowAlley(2);
 xml:
<Backgrounds place='alley' frame='2'/>

Common Parameters/Attributes for XML:

place
    - this is the location to show. There are a number of variations based on british and english spelling

mplace
    - this is place on the screen to show the background. By default all backgrounds are shown in the main window. This allows you to have the image shown in the person window, full screen, introduction windows etc. Fairly limited usefulness but available for use
eg
<Backgrounds place='alley' mplace='introduction'/>


General Notes:
  • when a background is shown any currently shown background or overlay is hidden first.
  • For night actions the image will either be darkened or if you do not specify frame then an appropriate image will be selected. Most indoor backgrounds do not do this as there is assumed to be lighting
  • <Background> or <backgrounds> or <background> can be used interchangably in xml
Thumbnails of all backgrounds are in the subfolder
Standard Images/Backgrounds
of this SDK. The number in the filename, like
tn_Background - Beach 5.jpg
is the frame number for use in the functions below (frame parameter/attribute)

A page showing the tumbnails is below
Thumbnails

SlaveModule Notes:

 Using events or slaves based on the SlaveModule class, a reference is provided to the Backgrounds object so _root is optional

eg
Backgrounds.ShowBedRoom();

function ShowAlcove(frame:Number, flip:Boolean)
XML:
<Backgrounds place='alcove'  frame='x' flip='boolean'/>

Shows a set of alcoves, like in a shop.

Note:
No night variants or effects.
frame parameter is unused, only a single image is provided

 eg
 actionscript:
_root.Backgrounds.ShowAlcove();
 xml:
<Backgrounds place='alcove'/>
 <Backgrounds place='acove' flip='true'/>

ShowBedRoom

actionscript:
function ShowBedRoom:Number, flip:Boolean)
XML:
<Backgrounds place='bedroom'  frame='x' flip='boolean'/>

Show a bedroom interior. If you omit all parameters it will always show the same image, the one chosen the represent your slaves bedroom. An image is selected based on your chosen house. The selected frame is (after the first call to this function)
_root.Home.HomeRoom
Change this value to select a different room, or just use the frame parameter

Note: No night variants or effects.

ShowOverlay

actionscript:
function ShowOverlay(colour:Number)
XML:
<Backgrounds colour='xxxxxx'/>
or
 <Backgrounds color='xxxxxx'/>

show a solid background colour in the main window

 colour - hex value like 0xff1010 (rgb format). Default 0 (white)
eg
_root.Backgrounds.ShowOverlay(0);             // solid black
_root.Backgrounds.ShowOverlay(0xFF0000);     // solid red
_root.Backgrounds.ShowOverlay(0xFFFFFF);     // solid white, but it is the default so not particularly needed

 eg
 actionscript:
_root.Backgrounds.ShowOverlay(0);             // solid black
_root.Backgrounds.ShowOverlay(0xFF0000);     // solid red
_root.Backgrounds.ShowOverlay(0xFFFFFF);     // solid white, but it is the default so not particularly needed
xml:
<Backgrounds colour='0xff0000'/>


 Special Versions
 actionscript:
function ShowOverlayWhite();
function ShowOverlayBlack();
XML:
<Backgrounds colour='white'/>
 <Backgrounds colour='black'/>

Show pure white or black background

actionscript:
function SetOverlayFromMovie(mc:MovieClip, offsetx:Number, offsety:Number)

Sets the background based on the colour of a pixel in a movieclip. Used so you can display a random movie clip and set the background colour based on the image.
