Contests
Slave Maker 3 SDK Updated for Version 3.4.01

There are a series of functions and variables associated with the contests that happen every 8 days.

All code and graphics are held in a class
var Contests:ContestsModule;
in _root scope

All images are held in a swf file
Engine/Contests.swf
and loaded into a MovieClip
_root.ContestsMovie
and also in the folder
Images/Contests
in the game installation.

Basic Concepts

Date/Time

The contests happen every 8 calendar days based on day of the year, not days elpased in training. They happen as a daily Event, starting at 8am and last 3 hours.

actionscript:
if ((_root.GameDate % 8) == 0) { // contest today }
xml:
<if date='contest'>

Rivals

All contests have 5 other competitors called Rivals, standardly referred to as Rival A, Rival B, Rival C, Rival D and Rival E. Not all contests use all 5 of these, but most do.
Each are assigned a default score generally in the range 0 to 200. Images for the rivals are mostly embedded in the Contests.swf file.

Score

All slaves are assigned a base score equal to their Reputation statistic divided by 10.

Rigged Contests

A contest can get rigged so the slave will ALWAYS win, standardly this is by a favour from Lady Farun. This is defined internally by Bitflag1, flag 19 being set
if (_root.BitFlag1.CheckBitFlag(19)) { // rigged }
<if flag1-19='true'>

Slave Images

These functions can be overloaded for a slave to show the image for the contest and optionally modify the score calculated for the slave.

XML Details

Only image nodes are implemented for contest images. There are no explicit event type nodes executed at the time the image is shown. There is the standard option of <Show> node for the image and there you could show an arbitrary image and manipulate the score for the slave.

All the nodes otherwise are standard image nodes in <Images> for the slave and ar commonly <Contesttype>
eg
<Images>
    <ContestBeauty>1</ContestBeauty>

    <ContestXXX>
       <Show>
          <if isfairy='true'>
             <ShowImage image='Images/Slaves/Ifurita/Contest  - XXX (Fairy).jpg'/>
          <else>if</else>
             <ShowImage image='Images/Slaves/Ifurita/Contest  - XXX' frame='random4'/>
           </if>
    </ContestXXX>
<Images>

Actionscript Details

The following image selection functions are used to show the image and alter the score for the slave if desired, Show an image appropriate for your slave for the contest. The parameter score is the standard calculated score for the contest. return the score with any changes you wish
eg
function ShowContestHousework(total:Number) : Number
{
         ClipContestsHousework._visible = true;
         return total;
}

eg Ranma is competitive so gets a score bonus
function ShowContestCourt(total) : Number
{
         _root.Backgrounds.ShowPalace();
         ClipContestsCourt._visible = true;
         return total + 8;
}

Other Slave Functions

These functions can be overloaded for a slave to control or customise the competitions. These are used by custom contests (see below) or to override specific rivals, or just totally alter the contest process. These can be implemented by slaves, assistants and events. They cannot be affected by Minor Slaves.

StartContest

 implemented in: SlaveModule class
actionscript:
function StartContest(num:Number, actno:Number)
XML:
Node: <Contests><StartContest>
variables:
general is set to Contests.NumContest
general2 is set to the act number for the contest

When this is called the contest is defined with all rival's scores set. Your slave's score is also set. Here you can alter the scores for rival's or your slave's calculated score. You could even abort the contest, but this is best done using DoContestNext

DoContestsNext

 implemented in: SlaveModule class
actionscript:
function DoContestNext(num:Number, actno:Number)
XML:
Node: <Contests><ContestsNext>
variables:
general is set to Contests.NumContest
general2 is set to the act number for the contest

num is the current value of _root.Contests.NumContest
return true to disable the standard processes and do your own thing. For the custom contests the values are 90+ there is no standard effect. Note the value of _root.Contests.NumContest is automatically incremented by 1 for every call. To end the contest change _root.Contests.NumContest to 1000

ContestBonus

 implemented in: SlaveModule class
actionscript:
function ContestBonus(num:Number) : Number
XML:
Node: <ContestsBonus>
this node is used from all of
<Slave>
<Assistant>
<Events>

num is the current value of _root.Contests.NumContest

 Return a bonus for the slave's score. The xml nodes are a parsed value
eg
<ContestBonus>

             <SetValue>100</SetValue>
       <else>if</else>
              <SetValue>0</SetValue>

</ContestBonus>

So to create a custom contest

a) In Initialise()
var cnum:Number = _root.AddCustomContest("Flower\rArranging", "She will compete in a contest of flower arranging and etiquette.");
_root.ShowCustomContest(cnum);

b) Implement StartContest

function StartContest(num:Number)
{
         if (num == 90) {
                  _root.Contests.RivalAScore = 243 + (_root.Difficulty * 5);
                  _root.Contests.RivalBScore = 310 + (_root.Difficulty * 5);
                  _root.Contests.RivalCScore = 194 + (_root.Difficulty * 5);
                  _root.Contests.RivalDScore = 141 + (_root.Difficulty * 5);
                  _root.Contests.RivalEScore = 115 + (_root.Difficulty * 5);
                  _root.Score = _root.Score + _root.VarBlowJob + _root.VarFuck;             // _root.Score is a base number based on reputation and a few other
         }
}

Note Pick whatever scores for your rivals and your slave as needed.
This is optional you can do anything or nothing here, as you like, or do everything on DoContestsNext

c) Implement DoContestsNext

Note: most of this is a variation of the code for the Lady of the Court contest, slightly adapted

function DoContestsNext(num:Number) : Boolean
{
         if (num == 90) {
                  // show first rival and show thier score. Note Backgrounds are automatically hidden before this function is called
                  ClipRivalA._visible = true;               // or use ShowMovie as always
                  _root.SetText("This girl got " + _root.Contests.RivalAScore + " points.");
                  return true;
         } else if (num == 91) {
                  // same for second girl, note hide last graphics.
                  ClipRivalB._visible = true;
                  ClipRivalA._visible = false;
                  _root.SetText("This girl got " + _root.Contests.RivalBScore + " points.");
                  return true;
         } else if (num == 92) {
                  // same for third girl
                  ClipRivalC._visible = true;
                  ClipRivalB._visible = true;
                  _root.SetText("This girl got " + _root.Contests.RivalCScore + " points.");
                  return true;
         } else if (num == 93) {
                  // same for fourth girl
                  ClipRivalD._visible = true;
                  ClipRivalC._visible = true;
                  _root.SetText("This girl got " + _root.Contests.RivalDScore + " points.");
                  return true;
         } else if (num == 94) {
                  // same for fifth girl
                  ClipRivalE._visible = true;
                  ClipRivalD._visible = true;
                  _root.SetText("This girl got " + _root.Contests.RivalEScore + " points.");
                  return true;
         } else if (num == 95) {
                  // handle end of contest and report who won
                  _root.Contests.NumContest = 1000;        // Contest over
         ClipRivalECourt._visible = false;
                  // Show your slaves image
                  CustomContestClip._visible = true;                // or use ShowMovie as always

                  // handle Lady Faruns favour, score already adjusted by 10000
                  if (_root.CheckBitFlag1(19)) _root.AddText("The contest was fixed so " + _root.SlaveName + " came ");
         else _root.AddText(_root.SlaveName + " obtained : " + _root.Score + "pts and came ");
                  _root.ClearBitFlag1(19);

                  // report placing. Note order here depends on what you set in StartContest()
                  if (_root.Score < _root.Contests.RivalBScore) {
                           _root.Diary.AddEntry(_root.SlaveName + " came last place in the Flower Arranging contest.");
                           _root.AddText("last place.\r\r");
                  } else if (_root.Score < _root.Contests.RivalEScore) {
                           _root.Diary.AddEntry(_root.SlaveName + "came fifth place in the Flower Arranging contest.");
                           _root.AddText("5th place.\r\r");
                  } else if (_root.Score < _root.Contests.RivalAScore) {
                           _root.Diary.AddEntry(_root.SlaveName + " came fourth place in the Flower Arranging contest.");
                           _root.AddText("4th place.\r\r");
                  } else if (_root.Score < _root.Contests.RivalCScore) {
                                   _root.AddText("3rd place.\r\rYou win 500GP");
              _root.Money(330);
                           _root.SMMoney(170);
              _root.VarJoy = _root.VarJoy + 5;
                           _root.WinContest = 3;
                           _root.Diary.AddEntry(_root.SlaveName + " placed third in the Flower Arranging contest.", false, 3);
                  }
                  else if (_root.Score < _root.Contests.RivalDScore)
                  {
                          _root.AddText("2nd place.\r\rYou win 700GP");
                                   _root.Money(525);
                          _root.SMMoney(175);
                          _root.VarJoy = _root.VarJoy + 10;
              _root.VarReputation = _root.VarReputation + 5;
                         _root.WinContest = 2;
                          _root.Diary.AddEntry(_root.SlaveName + " placed second in the Flower Arranging contest.", false, 2);
                  }        
                          else
                          {
                          _root.AddText("1st place!\r\rYou win 1500GP");
                          _root.Money(1125);
                           _root.SMMoney(375);
                          _root.VarJoy = _root.VarJoy + 10;
                          _root.VarReputation = _root.VarReputation + 10;
                          _root.VarRefinement = _root.VarRefinement + 10;
                           _root.WinCustom++;
                           _root.WinContest = 1;
                           _root.Diary.AddEntry(_root.SlaveName + " won the Flower Arranging Contest!", false, 1);
                          }
                  return true;
         }
}

Note:
You only need to set _root.Contests.NumContest once to end the contest. Do not change it otherwise
You may have as few or many rounds as you like, but there are only 5 standard rival variables provided. You can use your own if you want. The game does nothing with them for a custom contest.
See the division of money (3/4 slave, 1/4 slave maker)
            _root.Money(1125);
         _root.SMMoney(375);
Also see stat rewards
_root.WinContest = placing in contest, only 1-3
