General Concepts

Options can be set to limit or alter can content seen in the game.

All option data is help in a class instance Options, and also copied into the global scope in _root. All the options can be changed as required but most of them should not alter during a given game.

No functions are provided to change or check options, you just directly check or change the listed variables. Importantly XML does not allow you to change options at all, so you can only alter them with actionscript code. This was done as it is very rare to every have to change an option during game play so the extra code was not implemented for XML.

All options are automatically saved by the game when you alter them using the dialogue in the game. If a developer just changes a variable for an option then they ARE NOT SAVED. If needed in actionscript code you can change the variable, and then call
_root.Options.SaveGlobalData();

Core Variables

As noted these are all stored in the Options object. Unless otherwise noted all variables here are of type Boolean

Options for Content and Events

The following options are the ones that a developer would reasonably check for content of a slave or event. Please note above that XML does not allow you to change any options.
Actionscript Variable XML Variable

BadEndsOn

 badends
If true then game ending events can happen that mean your Slave Maker is gone and cannot continue.

Generally test this and do one form of an event and if false then do a 'softer' version of the event that allow you to continue the game.

BDSMOn

 bdsm
true if events involving BDSM elements are in the game. If false the Bondage training and Ponygirls traning and events are disabled

bCatgirlsCommon

 catgirlscommon
If true then catgirl encounters are much more common in the game (60% of the time)

DickgirlOn

 dickgirls
(Number) Defines if dickgirls can be encountered in the game
         1 = dickgirls in game
         0 = they are not
Note: the XML variable dickgirls is a Boolean, with values true or false
eg
<if dickgirls='true'>
   <!-- dickgirls are in the game -->
</if>

Note:
In general it is better to use the function IsDickgirlEvent() as it checks this flag for events. The only reason to use this flag is if you want an event that is always dickgirl related.and has no random chance to not happen.

DickgirlLesbians

 dickgirllesbians
If true lesbians consider dickgirls to be a woman and can fall in love with them.

DickgirlTesticles

 dickgirltesticles
If there is a choice show dickgirl images with testicles if this is true

Difficulty

 difficulty
The difficulty level selected for the game, varies
        - 1 = east
         0 = normal
         1 = difficult
         2 = hard

FurriesOn

 furries
It true anthropomorhhic furries available in the game. This does not include True Catgirls

IncestOn

 incest
Are events featuring incest in the game

NonHumansOn

 nonhumans
Can sexual encounters with non-human beings happen. This does not include tentacles, furries, elves, orc or similar. This is for substantially non-human beings, centaurs, mermaids, lamias, driders etc.

PonygirlsOn

 ponygirls
Are there ponygirls in the game. If false then ponygirl training is disabled

RapeOn

 rape
Can events featuring rape occur, including events where your slave or assistant or slave maker is raped

PregnancyOn

 pregnancy
Can yout slaves, or slavemaker, become pregnant. Also are there events featuring pregnancy in the game.

bSoundsOn

 soundson
true if sound effects are on
eg
<if soundson='true'>
   <!-- sound effects are enabled -->
</if>

Note:
In practice you will very seldom, if ever, need to use this variable. The standard game functions below all internally test it and do not play any sound if false.

TentaclesOn

 tentacles
(Number) Are the tentacle beasts in the game
         1 = tentacles in game
         0 = they are not

Note: the following variables are initialised from these settings
SMData.PonygirlAware If PonygirlsOn is true this is set to 0, otherwise -1

Options for Game configuration

The following options used by the game interally. It is unlikely that at developer will have to check these. Please note that these do not have an XML implementation to check.


Actionscript Variable

CombatDifficulty
The difficulty of combats in the game
         -1 = no combats
         0 = standard
         1 = hard
         2 = you lose

GlobalTentacleFrequency
percentage rate of tentacle events (0-100) on a daily basis

GlobalDickgirlFrequency
how often dickgirls are encountered (only valid if DickgirlOn == 1)
         0        = 20% of the time you encounter the dickgirl version of an event (if it exists)
         1        = 80% of the time you encounter the dickgirl version of an event (if it exists)
         2        = 100% of the time you encounter the dickgirl version of an event (if it exists)

TutorialOn
If true show tutorials during gameplay

FullscreenOn
If true the game will start in fullscreen mode

Clock24On
If true use a 24hr clock

MetricOn
If true use the metric system for measurements

StatIconsOn
If true show icnos for statistics changes

StatImagesOn
If true an image reflecting the training or state of the slave or slave maker is shown behind the statistics

bLimitSavesOn
If true you can only save the game on selected days (contest days)

bShowVanillaOn
If true then vanilla slaves and assistants are shown in the Slave Market

AllDickgirlXFOn
If true then all slaves may transform into dickgirls

DickgirlStartOn
If true all slaves will start the game transformed into dickgirls


Note: the following variables are initialised from these settings
TentacleFrequency frequency for the current game in progress, initially GlobalTentacleFrequency
DickgirlFrequency frequency for spontaneous dickgirl transformations, initially GlobalDickgirlFrequency


Content Filters and Selections

Selection Functions

There are a number of setting in Options that can affect what events appear and what slaves or assistants are available. Some of them have a rate of appearance if enabled.

In general do not use these options, instead use the functions/conditionals

ActionScript
function IsDickgirlEvent() : Boolean
function IsTentacleEvent() : Boolean
function Catgirls.IsCatgirlEvent() : Boolean

XML
dickgirlevent or isdickgirlevent
tentacleevent or istentacleevent
catgirlevent or iscatgirlevent
and generically
istrainingevent-catgirltraining

eg
if (_root.IsDickgirlEvent()) {
   // do a dickgirl thing here
}

<if isdickgirlevent='true'>
   <!-- do it here too -->
</if>

Slave/Assistant Selection

Based on these selections slaves or assistants can be excluded from the game. For instance Reisz is disabled is Tentacles are turned off. If you disable dickgirls a number of minor slaves are removed.

The is done via the SlaveGirlxx.txt file, add the following

Tentacles=true&Dickgirls=true&Furries=true&Pregnancy=true&NonHuman=true&Ponygirls=true

If any of these are set then it means the corresponding option MUST be turned ON for the slave/assistant to be usable.

Currently only the dickgirl flag is tested for Minor Slaves.
